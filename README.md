# Cruise Control System – Software Testing Project

## Overview
This academic project focuses on software testing and quality assurance activities applied to a Cruise Control System. The objective was to evaluate system behavior against specified requirements and identify potential functional issues.

## Project Scope
- Functional testing
- Boundary and edge-case testing
- Requirement-based testing
- System behavior validation

## Notes
- The system implementation code was provided as part of the course material.
- The project emphasis was on testing and verification, not system development.

## Deliverables
- Test documentation (PDF)
- Test scenarios and analysis

## Course
SWE326 – Software Engineering / Software Testing
